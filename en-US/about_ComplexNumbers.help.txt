SHORT DESCRIPTION
    ComplexNumbers allows you to work with the Numerics.Complex Objects

LONG DESCRIPTION
    ComplexNumbers allows you to work with the Numerics.Complex Objects

    You can find documentation here:
https://msdn.microsoft.com/en-us/library/system.numerics.complex(v=vs.110).aspx

DETAILED DESCRIPTION
    This is a rudimentary module that allows easy creation and manipulation of Systems.Numerics.Complex objects. 

    Objects returned from New-ComplexNumbers can the be displayed in human readable forms with Show-ComplexNumbers. 

    Complex math can be invoked upon the objects with Invoke-ComplexMath.

    For further assistance, each functions has its own documentation.
